Samvya-CRUD Library

Production-ready Spring Boot library that eliminates CRUD boilerplate for JPA (MySQL, PostgreSQL) and MongoDB

Core Features

  • Annotation-driven configuration
  • Compile-time validation
  • Auto-generated CRUD endpoints
  • Built-in caching (LRU)

Database Support

  • MySQL with dialect adapter
  • PostgreSQL with dialect adapter
  • MongoDB with aggregation
  • Multi-database support

Developer Experience

  • Zero boilerplate code
  • Automatic audit fields
  • Unique constraint handling
  • Comprehensive docs

Quick Start Example

// 1. Add dependency to pom.xml
<dependency>
    <groupId>com.samvya</groupId>
    <artifactId>samvya-crud-starter</artifactId>
    <version>1.0.0</version>
</dependency>

// 2. Annotate your Spring Boot application
@SpringBootApplication
@SamvyaCrud
public class MyApplication {
    public static void main(String[] args) {
        SpringApplication.run(MyApplication.class, args);
    }
}

// 3. Create your model
@Data
@SamvyaUniqueConstraint(fields = {"email"})
public class User extends SamvyaJpaEntity<Long> {
    private String name;
    private String email;
}

// 4. Create your controller (that's it!)
@RestController
@RequestMapping("/api/users")
public class UserController extends SamvyaCrudController<User, Long> {
    // All CRUD endpoints auto-generated!
}
